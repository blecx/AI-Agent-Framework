# AI-Agent-Framework

Dieses Projekt enthält mehrere Versionen von AI-Agenten, die jeweils in einem eigenen Ordner und einer eigenen virtuellen Umgebung gekapselt sind.

## Struktur

- Jeder Agent befindet sich in einem eigenen Unterordner (z.B. `openAI-generic`).
- Jeder Agent verwendet eine eigene virtuelle Umgebung.

## Erster Agent: openAI-generic

- Enthält die Basis für einen generischen OpenAI-Agenten.
- Die virtuelle Umgebung wird im jeweiligen Agent-Ordner erstellt.
- Das OpenAI-Framework ist installiert.

## Setup

1. Klone das Repository.
2. Wechsle in den gewünschten Agent-Ordner.
3. Aktiviere die virtuelle Umgebung und installiere die Abhängigkeiten.

## Weitere Agenten

Füge weitere Agenten in eigenen Ordnern hinzu und initialisiere jeweils eine eigene virtuelle Umgebung.
